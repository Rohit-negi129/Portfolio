Projects

Food Express

Full Stack Web Application | Angular, Spring Boot, MySQL

The Food Express is a full-stack food ordering system designed using Angular for the frontend and Spring Boot microservices for the backend. The backend architecture includes API Gateway, Eureka Server, Order Service, Restaurant Service, User Authentication, and User Service, ensuring a modular and scalable system. The application integrates MySQL and MongoDB for efficient data management. With a seamless UI and robust backend, The Food Express enables users to browse restaurants, place orders, and manage accounts effortlessly. This project showcases my expertise in microservices architecture, API integration, and full-stack development.
